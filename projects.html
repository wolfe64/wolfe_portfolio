Illegal Wildlife Trade Dashboard

Description

This dashboard illustrates illegal wildlife trafficking using four Tableau visualizations: a proportional dot map, a stacked bar chart, an anomaly detection graph, and a cycle plot. Filters allow exploration by taxa groups like amphibians and mammals. Key patterns such as country-specific preferences and biome hotspots are visible.

Analysis and Reflection

The dashboard is effective and interactive. Filters and animation help users explore patterns. One major insight: tropical biomes experience more trafficking. The cycle plot was harder to read and could be improved.

National Parks Recreational Visits: 2023

Description

This dashboard explores and ranks National Parks by 2023 recreational visits. It features a proportional dot map, a stacked bar chart, a tree map by region, and a line graph showing growth over time. Data was processed using Excel and QGIS.

Analysis and Reflection

This project was rewarding. Despite simple visualizations, the background data work (merging multiple datasets) was complex. One insight: Great Smoky Mountains National Park accounted for 14.39% of all visits nationwide.

Data Visualization LibGuide

Description

This LibGuide introduces beginners to data visualization concepts, including types of data, choosing the right tools, and links to further resources. It is designed for accessibility and ease of use.

Analysis and Reflection

The guide simplifies complex topics without overwhelming beginners. Given more time, I would expand it with more advanced examples and a section on critical data ethics.
