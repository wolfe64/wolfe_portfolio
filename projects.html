Illegal Wildlife Trade Dashboard

Description

For this example, I will be using a dashboard I created using the data visualization software Tableau.The issue that this dashboard illustrates is illegal wildlife trafficking. The dashboard contains four types of visualizations: a proportional dot map, a horizontal stacked bar chart, an anomaly detection graph, and a cycle plot. All of these visualizations except for the cycle plot can be filtered on four basic taxa groups: amphibian, bird, mammal, and reptile. However, the stack bar chart is filtered with a separate box as the data comes from a different data set. The proportional dot map illustrates the total recorded number of illegally traded wildlife by country based on the four taxa groups. The data in this visualization falls within the years 2000 to 2015. The size of each dot grows in accordance with the total quantity traded while the color illustrates the taxa. The anomaly detection graph uses the same dataset. However, it focuses on how many animals were traded each year and determines if any years were unusually high or low. It highlights low-quantity anomalies in blue and high-quantity anomalies in red. The horizontal stacked bar chart illustrates the number of wildlife trades according to the biome the animal is pulled from. Finally, the cycle plot looks for patterns in illegal wildlife trade based on the quantity traded each month from 2000 to 2015.

Analysis and Reflection

The dashboard effectively illustrates the issues surrounding illegal wildlife trafficking in an engaging and interactive way. Animation and filters allow users to immerse themselves in the issue and make changes to the data so they can make assumptions based on the visualizations. For example, the proportional dot map informs us which countries are importing the most wildlife and what types of wildlife they are most interested in. Using the filters, we can see how certain countries are more interested in certain animals than others. For example, South America seems to have a greater interest in reptiles compared to other taxa. From the stacked bar graph, we can see that warmer biomes have to deal with far more trafficking compared to colder ones across all taxa. The only visualization that could have been a bit better was the cycle plot, which is a bit difficult to read.


National Parks Recreactional Visits: 2023

Description

This dashboard takes a focus to examine and rank National Parks based on the number of their recreational visits in 2023. The dashboard contains four types of visualizations: a proportional dot map, a horizontal stacked bar chart, a tree map, and a line graph. The proportional dot map illustrates the rank and location of each National park denoting the number of recreational visits by the size of each dot. The horizontal stacked bar chart illustrates very similar data but in a more orderly list format that can be read more simply at the expense of not being able ascertain the location. The tree map analyzes which regions of the United States have the most recreational visits to their parks. Finally, the line graph simply displays how the number of recreational visits has grown since the founding of the National.

Analysis and Reflection

I really enjoyed working on this project. Firstly, I simply love the National Parks and I am happy to see the data trending so positively. Secondly, while the visualizations were all relatively simple there was quite a bit of data cleaning to be done behind the scenes to merge different data sets together. I had to use QGIS to convert a shapefile into a CSV to get access to the coordinates for the different national parks. I also had to use a number of lookup functions in excel to map data from different spreadsheets into one singular table.

Probably the most interesting observation to be made from the data is that the most popular National Park by a large margin is the Great Smoky Mountains National Park with 14.39% of all recreational visits in 2023. This is particularly interesting as the Southeast is ranked third as a region in recreational visits with 17.33% of all visits. This means the Great Smoky Mountains NP makes up 83% of visits in the Southeast in 2023. This is probably due to its location in the Appalachian Mountains and being near tourist attractions like Pigeon Forge.


Data Visualization LibGuide

Description

This work sample is a LibGuide I created on data visualization. LibGuides are a fantastic way to provide information services to not just members of a university but to anyone with access to the internet. This guide demonstrates my ability to provide information services to a wider audience and use analytical skills to assess sources and materials that fit the needs of patrons. In this LibGuide, I provide an overview of data visualization and its best practices. I wanted to aim this guide mostly at beginners. The LibGuide is divided into four major sections ignoring the introduction on the first page: Data 101, Tools, Inspiration, and Additional Resources. In the Data 101 section, I supply information on what to think about when picking a visualization method such as the different types of data, best uses of color, and color blindness. I also provide a few simple examples of common visualization as well as a link to a fantastic website where they can find more types of visualizations. The Tools section discusses the different types of software and programming languages that can be used to create data visualizations and how to pick the right one for you. The Inspiration section provides a variety of select articles and websites to inspire visitors and show them just how exciting and beautiful data can be when properly visualized. Finally, the Additional Resources section provides users with a list of select articles and books from the Thomas Cooper Library at the University of South Carolina which may be beneficial to anyone who wants to learn more about data visualization.

Analysis and Reflection

This project provides a simple yet informative overview of what it takes to make high-quality data visualizations. Each section provides users with beneficial information and resources. The guide also keeps the subject matter simple enough that it hopefully would not feel too overwhelming to a beginner. If more time was permitted, I would love to go back and add some more complicated data visualization examples.
